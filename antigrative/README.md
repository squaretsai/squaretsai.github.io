# YouTube Subtitle Downloader

一個現代化且簡潔的 YouTube 字幕下載工具，支援自動抓取影片字幕並在網頁上直接查看與下載。

## 功能特點
- 快速解析 YouTube 網址。
- 獲取影片原始字幕（支援手動與自動產生）。
- 極簡深色模式設計，支援響應式介面。
- 一鍵下載為 `.txt` 檔案。

## 安裝與運行

1. 安裝必要套件：
   ```bash
   pip install -r requirements.txt
   ```

2. 執行程式：
   ```bash
   python app.py
   ```

3. 開啟瀏覽器並輸入：`http://127.0.0.1:5000`
