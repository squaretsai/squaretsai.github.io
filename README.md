# 物語3 猜拳查詢（GitHub Pages 版）

## 使用方式
1. 把整個資料夾內容上傳到 GitHub repository 根目錄
2. 到 GitHub Repository → Settings → Pages
3. Source 選 `Deploy from a branch`
4. Branch 選 `main`，資料夾選 `/ (root)`
5. 存檔後等待 GitHub Pages 發佈
6. 用手機打開網址後，可加入主畫面使用

## 功能
- 即時搜尋魔物名稱或種族
- 點選後顯示「使用招式 / 對應招式」
- 支援手機版
- 支援 PWA，可加入主畫面
- 支援離線快取

## 資料來源
- 你匯出的 `mhs3_guessing_table.csv` 已轉成 `data.json`
